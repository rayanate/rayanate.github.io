Course Details

Web Application Programming and Hacking

Instructor: Dr. Phu Phung

Description: In this course, students will study basic web application development with front-end (HTML5, JavaScript, CSS) and back-end (PHP/MySQL). Web application vulnerabilities and attacks will be introduced and explored with hands-on exercises on the range. Secure programming principles and practices will be introduced to avoid potential web application vulnerabilities and attacks.

Course Curriculum

  • Lab 0 - Development Environment Setup

    In this Lab, we have setup virtual machine and software installations in the VM. The git repository setup is also done as part of this lab.

  • Lab 1 - Foundations of Web

    1. Familiar with the Wireshark tool and HTTP protocol. 2. Understanding HTTP using telnet and Wireshark. 3. CGI Web applications in C 4. A simple PHP Web Application with user input. 5. Understanding HTTP GET and POST requests

  • Lab 2.1 - Front-end Web development

    1. Basic HTML with forms, and JavaScript Developed a simple HTML file with basic tags, an image of your headshot, and a form. 2. Javascript code to display analog clock, display clock, show/hide email etc.

  • Lab 2.2 - Front-end development with JavaScript CSS, and jQuery

    1. Ajax, CSS, jQuery, and Web API integration Ajax, CSS, and jQuery handson has been performed in this lab.

  • Hackathon 1 - Cross-site scripting attacks and defenses

    This hackathon consists of multiple hands-on exercises, where you would perform cross-site scripting attacks on multiple levels to understand the vulnerabilities as well as defense methods.
